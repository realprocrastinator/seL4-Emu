Physical Memory Emulation Implementation

To implement this we are going to use the shm_open to create a shared memory and use this as our emulated physical memory as well as the 1:1 physical mapping area in the kernel window

Therefore, we also solved the challenge of passing bootinfo to the roottask as well as accessing IPC buffer.

  • to pass the bootinfo, we first locate the bootinfo frame by calculating the offset.
  • pass the offset using emulation internal IPC to the seL4 application.
  • seL4 application maps a page for the part of the file based on the offset.
  • the same method makes mapping IPC buffer work as well.
